Here is a list of my Projects

  • Needy

    Needy was the first real android project I build. It was a part of a university course where we were suppose to build an andrioid application for truck drivers. The functionality was very basic, it had a starting screen with four options, were a truck driver could select either: Food, Reststop, Gasstation or settings. I was in charge of mining JSON data from different official and no-nofficial (reverse engineering websites) to get the information about near reststops and gasstations. We also used the Google maps API to get near restaurants and to get directions to the nearest.

  • Pling

    The pling project was another university project where a few classmates and I build a simple order system for a local bike delivery company. The system included a website to create orders and an android application for the bikers to get directions and todays orders. I was in charge of the backend, this was done without any external librarys, in PHP. I managed the website and database with another person in charge of the front-end look.
